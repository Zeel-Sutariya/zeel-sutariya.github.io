Projects 

  • GUI simulation of Card Game using C# - WPF

    As part of my final semester project, I worked in a team to create a GUI simulation of Tarneeb, a popular Middle Eastern card game, using C#- WPF.

    Our project involved designing and implementing the game rules, including the Bidding System, Card Dealing, Trump Suit Selection, and Game AI (Easy, Medium, and Hard).

    We also incorporated various variants of Tarneeb into our simulation. Through this project, I gained valuable experience in software development, problem-solving, and teamwork.

    Skills:  C#, GUI, WPF

  • Hospital Database Development and CRUD prototype

    I created a database for a hospital and developed a prototype of CRUD (Create, Read, Update, Delete) functionality using SQL and C#- WPF. The database was designed to store patient information, medical history, and treatment plans.

    I implemented various features such as data validation, search functionality, and user authentication to ensure data security and accuracy.

    This project allowed me to apply my knowledge of database design and development, as well as gain experience in implementing user interfaces and integrating database systems with software applications.

    Skills:  C#, SQL, Windows Software Development

  • Responsive Web and Mobile Application for Accommodations

    As part of a final group project, I played a key role in the development of a responsive web and mobile application designed to help international students find accommodations.

    Following Agile methodology, I was responsible for creating a project timeline, estimating total costs, sprint planning, sprint review, and daily scrums.

    Additionally, I contributed to the development and improvement of the application, including sprint planning, design, coding, testing, and demos.

    I also wrote automated tests and helped define user stories and UI/UX requirements. Through this project, I gained valuable experience in project management, software development, and team collaboration.

    Skills:  Java, CSS, JavaScript, SQL
